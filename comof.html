Como funciona

Basicamente, la cosa es simple: la pieza clave es el "browser","navegador"."visualizador" o "cliente" o como quieras llamrle, HTML, es decir, el programa que ahora mismo estamos utilizando para ver esto. Todas las codificaciones de efectos en el texto que forman el lenguaje HTML no son mas que instrucciones para el visualizador. Partiendo de esto, se netiende el porque no se ve lo mismo con todos los visualizadores. Depende de como esten diseñados y para que version de lenguaje esten diseñados.
